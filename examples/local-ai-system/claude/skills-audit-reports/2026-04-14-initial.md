## Findings

### P1

- 问题：`~/.claude/skills/llm-wiki/scripts/lint_wiki.py` 使用了 `dict | None` 类型注解，要求 Python 3.10+；本地默认 `python3` 是 3.9.6，导致 lint 直接崩溃
- 影响：`llm-wiki` 的核心治理动作 `lint` 当前不可执行，首次搭好的 `knowledge-wikis` 无法按预期做健康检查
- 证据：
  - `python3 --version` → `Python 3.9.6`
  - 运行 `python3 .../lint_wiki.py <wiki-root>` 报 `TypeError: unsupported operand type(s) for |: 'type' and 'NoneType'`
- 建议动作：
  - 方案 A：把脚本类型注解改为 Python 3.9 兼容写法（如 `Optional[dict]`）
  - 方案 B：明确要求本地运行 Python 3.10+，并在 `llm-wiki` 文档中写明
- 修复状态：已修复。`lint_wiki.py` 已改为 Python 3.9 兼容写法，并通过本地实际运行验证。
