# Role Routing

## Definition

Role routing 是在执行前先判断“谁应主导这次任务”的过程。

## Current Roles

- `dev`
- `pm`
- `ui`
- `arch`
- `ba`
- `sm`

## Routing Principle

先确定主角色，再决定是否需要其他角色提供辅助视角。  
不要在任务开始时就把多个角色混成一个没有边界的执行体。
